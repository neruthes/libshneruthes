# libshneruthes

Portable library of personal shell scripts.


## Usage

Install as repo dependency:

```sh
yarn add github:neruthes/libshneruthes
```

Add to PATH in make.sh:

```
export PATH="$PATH:$PWD/node_modules/libshneruthes/bin"
```
